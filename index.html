Spatial Analysis

Toronto's Primary Care Enrolment

Statistical analysis conducted in ArcGIS Pro

This cartographic output summarizes the method and results of a statistical analysis of Primary Care Enrolment in Toronto, Ontario. The variables involved in the analysis were: employed population, university graduates with a certificate or diploma, median household income after tax, and primary care enrolment values. Data related to these variables were gathered through reliable open-sourced hubs. ArcGIS Pro's Ordinary Least Squares and Geographically Weighted Regression tools were used to conduct and evaluate the analysis.

Driving Time to Toronto's Health Services

Notebook created in ArcGIS Pro

This is one page of a 196-page ArcGIS Pro Notebook of the City of Toronto shows the proximity of Substance Use Treatment Locations to Ambulance Stations within the city by vehicle. Closest Facility Network Analysis tools were used to find ambulance stations within 4 minute drive time to the substance use treatment centers.

Toronto's Health Services Proximity to TTC

Interactive ArcGIS Dashboard

This dashboard shows the locations of Substance Use Treatment Centres and Ambulance Stations within Toronto, ON, and their proximity to the city's TTC route. ArcGIS Pro's Linear Referencing and Geocoding tools were used to create the layers. Layers were then shared to ArcGIS Online, where they were added to a Dashboard application.

Snapping Turtle Habitat Suitability Analysis

Interactive ArcGIS StoryMap

To potentially help monitor and protect the snapping turtle species, an analytic hierarchy process (AHP) multi-criteria decision analysis (MCDA) was performed to determine areas most suitable for their habitat. The 5 criteria considered in this analysis were Aspect, Land Cover, Distance from Waterbody, Distance from Road, and Distance from Fishing Access Points. Data related to these variables were gathered through reliable open-sourced hubs. In the StoryMap, users are able to compare the original layers to the site suitability layers, as well as explore the resulting suitability map.

Environmental Modelling

Picnic Site Suitability Analysis

3D Local Scene created in ArcGIS Pro

A simple terrain analysis was conducted to find the most suitable areas for a picnic in the Peterborough Area of Ontario. The criteria considered in the analysis were slope, aspect, proximity to infrastructure, and proximity to natural phenomena including parks and vegetation. 3D visualization was added to the cartographic output using ArcGIS Pro's Local Scene to provide an additional perspective to one of the most suitable areas for a picnic.

Comparing Drainage Lines

Using ArcHydro tools in ArcGIS Pro

ArcHydro tools within ArcGIS Pro were used to create a map comparing drainage lines. Multiple ArcHydro Terrain Preprocessing tools were applied to the DEM derived Flow Direction to create Drainage Line 1. Similar tools were applied to the Enhanced Flow Direction Grid to create Drainage Line 2, a hydrology dataset based off provincial hydrology and elevation data. These two sets of drainage lines were compared visually to each other and the original drainage network data.

Remote Sensing

Land Cover Across the Oak Ridges Moraine

Classified in PCI CATALYST and symbolized in ArcGIS Pro

This map shows the land cover across the Oak Ridges Moraine. The remote sensing data were processed in CATALYST Professional, where the six classes of land cover were derived using the supervised maximum likelihood classification method. They were then brought into ArcGIS Pro to symbolize and create a cartographic output with various map elements.

LiDAR Point Cloud Analysis

Using LAS datasets in ArcGIS Pro

This figure compares the results of a LiDAR Point Cloud Analysis of the Bailieboro area in Southern Ontario - DTM, DSM, and original True Color Composite imagery. ArcGIS Pro was used to convert, process, filter, and symbolize the LAS Datasets. The layers were symbolized using Shaded Relief to show elevation. View with 3D ChromaDepth glasses to achieve a 3D effect on the DTM and DSM.

Enhancing Aerial Images

Using algorithms and tools in PCI CATALYST and ArcGIS Pro

Various enhancement techniques performed on USGS EarthExplorer Landsat and SRTM images. The remote sensing images were processed and enhanced in PCI CATALYST Professional. Then they were brought into ArcGIS Pro to be enhanced further, annotated, and laid out to create a cartographic output with various map elements.
